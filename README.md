# Nail Salon Lumière

「Nail Salon Lumière」は、  
**上品・ミニマル・静かな高級感**をコンセプトに制作した、ネイルサロン向けの1ページ型ランディングページ（架空案件）です。

ポートフォリオ掲載を前提に、  
ブランド世界観の設計からUIデザイン、HTML/CSSコーディング、レスポンシブ最適化まで一貫して制作しています。

---

## 制作意図

- 大人女性向けサロンに必要な「清潔感」「信頼感」「落ち着き」を、白・ベージュ・ブラウン系のトーンで表現
- 余白・タイポグラフィ・光の演出を活かし、“静かな上質感” を感じられるUIを設計
- 初見ユーザーが迷わず予約導線へ進める構成を重視

---

## ターゲット

- 20代後半〜40代の働く女性
- 派手すぎない、上品で日常になじむネイルを求める層
- サロンの雰囲気や丁寧さを重視して比較検討する層

---

## 工夫したポイント

### HERO設計
- コピー・余白・視線誘導を整理し、第一印象で世界観が伝わる構成に設計
- 背景グラデーションと光表現を組み合わせ、柔らかい空気感を演出
- デバイスごとに余白バランスを最適化し、スマホでもブランド感を維持

### UI / デザイン
- ガラス調カード（Glassmorphism）を使用し、透明感と上品さを表現
- セクションごとに背景トーンを微調整し、サイト全体の色温度を統一
- hover演出やフェード表現を最小限に抑え、静かなエディトリアルデザインを意識

### 情報設計
- Concerns → Solutions → Why → Menu → Reviews の流れで、心理的ハードルを下げる構成
- MENUはPCでは表形式、スマホではカード形式へ切り替え、読みやすさを最適化

### レビューセクション
- Swiper.js を使用したカルーセルを実装
- 中央カードを強調し、左右カードをフェードで自然につなげることで、上品な閲覧体験を演出
- スマホ・タブレットでも操作しやすいレイアウトへ調整

### Instagramセクション
- 横スクロールアニメーション（Marquee）を実装
- hover可能デバイスのみ停止する仕様にし、スマホでは自然に流れ続ける設計に調整

### レスポンシブ対応
- タブレット: `max-width: 959px`
- スマホ: `max-width: 559px`
- HERO / WHY / MENU / FLOW / REVIEWS / INFO を中心に、余白・文字サイズ・表示順を端末ごとに最適化
