# Nail Salon Lumière

「Nail Salon Lumière」は、**上品・ミニマル・やわらかな高級感**を軸に制作した、ネイルサロン向けの1ページ型ランディングページ（架空案件）です。
ポートフォリオ掲載を前提に、世界観設計からUI実装、レスポンシブ最適化まで一貫して制作しています。

## 制作意図
- 大人女性向けサロンに必要な「清潔感」「信頼感」「落ち着き」を、白・ベージュ・ブラウン系のトーンで表現。
- 情報量が多いLPでも、余白とタイポグラフィで視認性を担保。
- 初見ユーザーが迷わず予約導線へ進める構成を重視。

## ターゲット
- 20代後半〜40代の働く女性
- 派手すぎない、上品で日常になじむネイルを求める層
- サロンの雰囲気や丁寧さを重視して比較検討する層

## 工夫した点
- **ファーストビュー設計**: コピー・視線誘導・CTAの優先順位を整理し、第一印象で価値が伝わるように設計。
- **セクションの情報設計**: Concerns → Solutions → Why → Menu → Reviews の流れで、心理的ハードルを下げる構成。
- **MENU表示最適化**: PCでは表形式、スマホではカード形式へ切り替え、読み順（メニュー名→説明→価格）を最適化。
- **可読性調整**: 文字間・行間・コントラストを端末幅ごとに微調整し、上品さを保ちながら読みやすさを担保。
